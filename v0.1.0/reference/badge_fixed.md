# Fixed badge

Fixed badge

## Usage

``` r
badge_fixed(id, label, content)
```

## Arguments

- id:

  (`character(1)`) shiny module's id

- label:

  (`shiny.tag`) Label displayed on a badge.

- content:

  (`shiny.tag`) Content of a drop-down.

## Value

A `shiny.tag` object representing a fixed badge component to display the
selected option.
